Bootstrap Components Used

Layout & Navigation
  • Navbar: Responsive navigation with collapse functionality
  • Container: Responsive fixed-width containers
  • Grid System: Rows and columns for responsive layout
  • Flexbox Utilities: d-flex, align-items-center, justify-content-center
Components
  • Cards: Multiple card layouts with images and content
  • Carousel: Image sliders with indicators and controls
  • Modal: Contact information popup dialog
  • Buttons: Various button styles and sizes
  • Badges: Grade indicators with color coding
  • Alerts: Information display boxes
Styling & Utilities
  • Typography: fw-bold, text-center, lead classes
  • Spacing: py-5, mb-3, g-4 for margins and padding
  • Colors: bg-light, bg-dark, text-white, text-muted
  • Shadows: shadow-sm for card depth
  • Rounded: rounded, rounded-circle for image styling
  • Height: h-100 for equal height cards
Icons
  • Bootstrap Icons: bi-envelope, bi-telephone, bi-linkedin, bi-download, etc.
  • Icon Sizes: fs-1, fs-2 for different icon sizes

Dynamic Features & JavaScript

Bootstrap JavaScript Features
  • Carousel Auto-play: Automatic image rotation
  • Navbar Collapse: Mobile menu toggle
  • Modal Functionality: Contact info popup
  • Carousel Controls: Next/previous navigation
  • Carousel Indicators: Slide position indicators
Custom JavaScript
  • Hero Image Animation: Floating effect on page load
  • Parallax Scrolling: Hero section background movement
  • Opacity Effects: Fade effects during scroll
  • Event Listeners: Load and scroll event handling

Responsive Design Features

Breakpoint Classes
  • col-md-6: Two columns on medium+ screens
  • col-lg-4: Three columns on large+ screens
  • order-md-1/2: Column reordering on different screens
  • text-md-start/end: Text alignment changes
Mobile Optimization
  • Navbar Toggle: Collapsible mobile menu
  • Responsive Images: img-fluid class
  • Flexible Cards: Stack on mobile devices
  • Touch-friendly: Large buttons and touch targets

  • images/: All portfolio images

External Dependencies
  • Bootstrap 5.3.0: CSS framework

  • Bootstrap JS: Interactive components
